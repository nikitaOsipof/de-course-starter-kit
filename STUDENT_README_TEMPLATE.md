## 🚀 О проекте
В данном репозитории разрабатывается сквозной производственный пайплайн (Data Pipeline). Каждая лабораторная работа оформляется в отдельной Git-ветке и сливается в ветку `main` после успешного прохождения автоматических тестов (CI/CD).

## 🛠️ Инструкция по локальному запуску (Для проверяющего)
1. Склонировать репозиторий.
2. Создать окружение: `uv venv --python 3.11`
3. Установить зависимости: `uv pip install -e .[dev]`
4. Запустить тесты: `pytest tests/`
